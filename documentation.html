Devuelve las terminales del comercio autenticado (se toma de Authorization: Bearer <token>). Puedes filtrar por número de serie (búsqueda parcial y case-insensitive), nombre o ubicación (location).

Authorizations:
CommerceAuth
query Parameters
~serialNumber
string
Example: ~serialNumber=8214

Búsqueda parcial (regex, case-insensitive) por número de serie de la terminal. Usa coincidencia contiene. Ejemplo: ~serialNumber=08214 devuelve terminales cuyo SN contiene "08214".

name
string
Example: name=Ingenico Desk5000

Filtra por nombre exacto de la terminal.

location
string
Enum: "WAREHOUSE" "FRANCHISE" "COMMERCE"
Example: location=COMMERCE

Ubicación de la terminal.

limit
integer [ 1 .. 200 ]
Example: limit=10

Límite de resultados (por defecto 10).
